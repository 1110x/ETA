기성 LIMS(LabVantage, STARLIMS 등)와 ETA 비교.

**Why:** 릴리즈 목표 설정 및 우선순위 판단 기준으로 활용

**How to apply:** 부족한 항목 중 실사용 빈도 높은 것부터 우선 개발

| 항목 | 기성 LIMS | ETA 현재 |
|------|-----------|----------|
| 시험성적서 | 완성 | 70% |
| 견적/의뢰 관리 | 완성 | 60% |
| 직원/권한 관리 | 완성 | 50% |
| 채수/샘플 추적 | 완성 | 40% |
| 분석결과 입력/연동 | 완성 | 20% |
| 장비 연동 | 완성 | 5% |
| 규제/인증 대응(ISO 등) | 완성 | 10% |
| 통계/리포트 | 완성 | 15% |
| **우리 업무 흐름 맞춤** | 낮음 | 100% |

**전체 기능 기준:** 약 25~35% 수준  
**실사용 기준 (우리 회사에 필요한 기능만):** 약 70~80% 커버

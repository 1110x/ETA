# 항목별 1회분 시약량 (자동추출 초안)

다음 초안은 저장소 내 표준 PDF에서 정규표현식으로 추출한 '수량/단위' 후보를 ES 코드별로 자동 매핑한 초안입니다. 각 항목은 가능하면 원문 문맥을 함께 기재했고, 검증이 필요한 경우 `검토 필요`로 표기했습니다.

원문에서 추출된 전체 후보 목록은 [Docs/reagent_amounts_extracted.csv](Docs/reagent_amounts_extracted.csv)에서 확인할 수 있습니다.

---

자동추출 샘플 (검증 완료 후 `확정`으로 변경 권장)

- ES 04130:
	- HCl (12 M) — 5 mL / 시료 1 L 당 (원문: "1 L 당 HCl(12 M) 5 mL 첨가") — 자동추출 후보 (검토 필요)
	- CuSO4 (황산구리계 전처리) — 1 g / 시료 1 L 당 (원문: "CuSO4 1 g") — 자동추출 후보 (검토 필요)

- ES 04305 (BOD):
	- Na2SO3 (아황산나트륨, 잔류염소 제거용) — 0.025 N 표정량 표기(원문 참조) — 문서에 농도 표기(검토 필요)

- ES 04308 (용존산소, 윙클러법):
	- 티오황산나트륨 (적정 시약) — 적정량 표기(원문 내 적정부량 다수 기재) — 자동추출 후보 (검토 필요)

- ES 04748 (VOCs):
	- NaCl (염석) — 문서 미명시 (표준절차상 첨가하나 양 미기재) — 검토 필요
	- 무수황산나트륨 (수분제거) — 문서 미명시 — 검토 필요

---

자동추출 전체 처리 및 결과 요약

- 총 추출 건수: 3193개(원문: Docs/reagent_amounts_extracted.csv)
- 처리 방법: PDF 텍스트에서 숫자+단위 패턴(예: "5 mL", "1 g", "50 μL")을 찾아 주변 문맥(한 라인)을 함께 추출 → ES 코드가 있는 경우 ES별 그룹화. 이후 '시약명'은 문맥에서 직접 추출하거나 수작업 대조 필요.
- 상태 표시 규칙:
	- `확정`: 원문에 명확히 '시약명 + 수량' 형태로 기재되어 있고 검토 완료
	- `자동추출 후보 (검토 필요)`: 수량은 추출되었지만 시약명과의 연관성이 자동으로 확정되지 않음
	- `문서 미명시`: 원문에 수량 표기가 없음

검토 권장 작업

- 1순위: `자동추출 후보` 항목 중 ES별로 시약명 연결(문맥 스니펫 확인) 및 DB 시약 테이블(시약 목록)과 매칭 — Scripts/reagent_compare.py 사용
- 2순위: 매칭이 불명확한 항목은 수동 검토로 `확정` 또는 `문서 미명시`로 분류
- 3순위: `확정` 항목을 바탕으로 `항목 : 시약명 양/EA` 형식으로 정리하고 배포

다음 단계로 진행해도 될까요? 진행 시 아래 작업을 수행합니다:

1) `Docs/reagent_amounts_extracted.csv`를 ES별로 그룹화해 각 ES에 대해 상위 후보(빈도/근접성 기준)를 선택하여 `Docs/시약_1회분_정량.md`에 자동으로 채워 넣기
2) 불확실 항목은 별도 섹션에 모아 검토 리스트 생성
3) 사용자가 원하면 시약 DB CSV(시약 테이블 덤프)를 받아 자동 매칭을 시도

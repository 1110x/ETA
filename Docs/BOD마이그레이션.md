## 자동 생성값

| DB 컬럼 | 값 | 비고 |
|---------|-----|------|
| 구분 | SN 접두사 판별 | 없음→여수, [율촌]→율촌, [세풍]→세풍 |
| 소스구분 | `"폐수배출업소"` | 고정값 |
| 등록일시 | NOW() | 삽입 시각 |
| 의뢰명 | `""` | 비움 |
| 시료분석 | `""` | 비움 |
| 비고 | `""` | 비움 |

## SN 처리 규칙

- `1-9-1` → NormalizeSN → `01-09-01`, 구분=`여수`
- `[율촌]1-9-1` → 접두사 제거 → `01-09-01`, 구분=`율촌`
- `[세풍]1-9-1` → 접두사 제거 → `01-09-01`, 구분=`세풍`
- 숫자만인 경우 (예: `37104`) → skip

## UPSERT 기준

- **중복 체크**: `SN + 업체명`
- 존재하면 UPDATE, 없으면 INSERT

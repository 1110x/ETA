# 생태독성 통계분석 시스템

## 개요

ES 04704.1c「물벼룩을 이용한 급성 독성 시험법」에 따른 생태독성 분석 시스템.
물벼룩(Daphnia magna) 24시간 급성독성시험 데이터로 EC50을 산출하고, TU(=100/EC50)로 변환하여 저장한다.

두 가지 통계 방법을 동시 지원:
- **TSK (Trimmed Spearman-Karber)** — 비모수적, 유영저해/사망 데이터 1개 이상이면 사용 가능
- **Probit** — 모수적(정규분포 가정), 1~99% 반응 데이터 2개 이상 필요

## 시험 기준 (ES 04704.1c)

### 시험 조건
| 항목 | 기준 |
|---|---|
| 시험생물 | Daphnia magna Straus (물벼룩) |
| 시험개체 | 생후 24시간 미만 어린 개체 |
| 시험시간 | 24시간 |

| 시험온도 | (20 ± 2) ℃ |
| 조명 | 명:암 = 16:8시간 |
| 시험용액량 | 50 mL |

### 표준 시험 설계
| 농도 | 시료 농도(%) | 반복구 | 생물수 |
|---|---|---|---|
| 대조군 | 0 (희석수만) | 4반복 × 5마리 | 20마리 |
| 1단계 | 6.25 | 4반복 × 5마리 | 20마리 |
| 2단계 | 12.5 | 4반복 × 5마리 | 20마리 |
| 3단계 | 25.0 | 4반복 × 5마리 | 20마리 |

| 5단계 | 100.0 (원수) | 4반복 × 5마리 | 20마리 |

### 판정 기준
- **치사**: 시험용기를 두드리고 15초 후 움직임이 없는 상태
- **유영저해**: 부속지를 움직이더라도 유영하지 못하는 상태
- **EC50**: 투입 생물의 50%가 치사 또는 유영저해를 나타낸 농도(%)

### QA/QC
- 대조군 사망률 **10% 초과 시 시험 무효**
- 표준독성물질(K₂Cr₂O₇) 시험: 24시간-EC50 = (0.9 ~ 2.1) mg/L 범위
- 표준독성물질 시험은 **월 1회 이상** 수행

## 결과 산출

### 통계적 방법으로 EC50 산출 가능한 경우 (7.5항)

```
TU = 100 / EC50
```

EC50 단위는 %, TU는 무단위.

### EC50 산출 불가 시 (8.1.2항)

| 100% 시료 유영저해율 | TU 계산 |
|---|---|
| 0 ~ 10% | TU = 0 |
| 10 ~ 49% | TU = 0.02 × 유영저해율(%) |
| 51 ~ 99% | 100%~50% 사이 추가 희석 필요 |
| 100% | EC50 < 100%, 통계적 방법 적용 |

예시: 100% 시료에서 20마리 중 5마리 유영저해 → 유영저해율 25% → TU = 0.02 × 25 = **0.5**

### 통계 방법 선택 기준 (7.6항)

| 방법 | 적용 조건 |
|---|---|
| Probit | 1~99% 사이 유영저해/사망 데이터 **2개 이상** |
| TSK | 유영저해/사망률 자료 **1개 이상** |

95% 신뢰구간 산출 불가 시 (100%와 0% 데이터만 있을 때):
상용로그 농도 vs 치사/유영저해 그래프에서 EC50 산출

## 알고리즘 상세

### TSK (Trimmed Spearman-Karber)

```
입력: 농도 C₁<C₂<...<Cₙ, 각 농도별 생물수 nᵢ, 사망수 rᵢ
      대조군 생물수 n₀, 사망수 r₀

1. 반응비율:  pᵢ = rᵢ / nᵢ

2. Abbott 보정 (대조군 사망률 Pc = r₀/n₀):
   pᵢ' = (pᵢ - Pc) / (1 - Pc)
   0 ≤ pᵢ' ≤ 1 로 클램핑

3. 단조증가 보정 (PAVA — Pool Adjacent Violators Algorithm):
   p₁' ≤ p₂' ≤ ... ≤ pₙ' 이 되도록 인접 위반 구간을 평균으로 교체

4. 자동 Trim 결정:
   trim = max(p₁', 1 - pₙ')

5. Trimmed 비율 조정:
   adj(p) = 0           if p < trim
          = 1           if p > 1-trim
          = (p-trim)/(1-2·trim)  otherwise

6. Spearman-Karber 추정:
   m = Σᵢ [(log₁₀Cᵢ₊₁ - log₁₀Cᵢ) × (adjᵢ₊₁ + adjᵢ) / 2]
   log₁₀(EC50) = log₁₀(Cₙ) - m

7. 분산:
   Var(m) = Σᵢ [dᵢ² × pᵢ(1-pᵢ) / (nᵢ-1)]
   SE = √Var(m)

8. 95% 신뢰구간:
   EC50_하한 = 10^(log₁₀EC50 - 1.96×SE)
   EC50_상한 = 10^(log₁₀EC50 + 1.96×SE)

9. TU = 100 / EC50
```

### Probit

```
입력: 동일

1. 반응비율 + Abbott 보정 (TSK와 동일)

2. 에지케이스 처리 (Miller-Tainter):
   p = 0  →  p = 0.25/n
   p = 1  →  p = (n-0.25)/n

3. Probit 변환:
   Y = Φ⁻¹(p) + 5
   여기서 Φ⁻¹ = 표준정규분포 역함수

4. 가중최소제곱 회귀 (IWLS, 최대 25회 반복):
   모델: Y = a + b × log₁₀(C)
   가중치: w = n × p × (1-p)

5. EC50 계산:
   probit = 5일 때: 5 = a + b × log₁₀(EC50)
   log₁₀(EC50) = (5 - a) / b
   EC50 = 10^((5-a)/b)

6. 95% 신뢰구간 (분산-공분산 행렬):
   V = (X'WX)⁻¹
   SE(log₁₀EC50) = √[V₁₁/b² + ((5-a)²/b⁴)·V₂₂ - 2(5-a)/b³·V₁₂]
   EC50 ± = 10^(log₁₀EC50 ± 1.96×SE)

7. TU = 100 / EC50
```

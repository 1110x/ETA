# 시약 정리 - 총괄

이 문서는 `Docs/시약정리_2일반항목.md`, `Docs/시약정리_3이온류.md`, `Docs/시약정리_4금속류.md`, `Docs/시약정리_5유기물질.md`의 시약 항목을 통합한 요약입니다. 자동 비교 및 재고/소모량 산정의 기초로 사용합니다.

## 핵심 카테고리별 시약

- **산류**: 황산(H2SO4), 염산(HCl), 질산(HNO3)
- **염기류**: 수산화나트륨(NaOH), 수산화칼륨(KOH), 암모니아수(NH4OH)
- **표정·적정 시약**: 티오황산나트륨(Na2S2O3), 요오드산포타슘(KIO3), 과망간산칼륨(KMnO4), 옥살산나트륨(Na2C2O4)
- **산화제·분해제**: 과황산염(K2S2O8), 과산화수소(H2O2), 알칼리성 황산포타슘
- **완충/발색 시약**: 몰리브덴산암모늄+아스코빈산(총인), 설파닐아마이드/NED(아질산), DPD(잔류염소), 메틸렌블루(계면활성제)
- **용매(유기물질 분석)**: 노말헥산(n-hexane), 다이클로로메탄(CH2Cl2), 메탄올(CH3OH), 아세톤, 클로로폼, MIBK
- **건조·정제제**: 무수황산나트륨(Na2SO4), 실리카겔, 플로리실
- **전용시약·특수시약**: NaBH4, SnCl2, APDC, EDTA, NaN3, ATU, TCMP
- **기기 관련 시약/시스템**: HPLC급 정제수(18 MΩ), 아세토나이트릴, 메탄올, 용리액/억제기 재생액(이온크로마토그래피)
- **표준/내부표준**: 금속 표준(1000 mg/L), 유기 표준(PCBs, DEHP 등), 동위원소 내부표준(13C, d-표준 등)

## 사용처/비고
- 각 항목은 세부 시험방법(예: ES 04308 용존산소, ES 04400 금속류 등)에 대응합니다. 원문 PDF/MD를 우선 참고하세요.
- 본 총괄은 자동 매칭의 기준 문자열로 사용됩니다. 정확한 매칭을 위해서는 `품목명`의 전처리(단위 제거, 한글/영문 혼용 정리 등)가 필요합니다.

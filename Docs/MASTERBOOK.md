## 1. 시스템 개요

**ETA**는 수질분석센터의 업무 전반을 관리하는 데스크톱 애플리케이션입니다.

| 구분 | 내용 |
|------|------|
| 플랫폼 | Windows / macOS |
| 주요 기능 | 직원관리, 계약업체, 견적/의뢰서, 시험성적서, 보수요청, 물품구매, 폐수배출업소 |
| 데이터베이스 | 로컬 SQLite 또는 서버 MariaDB (선택) |
| 출력 | Excel(xlsx), PDF |

---

## 2. 설치 및 시작

### 2-1. 최초 실행

프로그램을 처음 실행하면 필요한 구성 요소가 있는지 자동으로 확인합니다.  
안내 창이 뜨면 **설치 시작** 버튼을 클릭하세요.

### 2-2. 데이터베이스 파일

로컬(SQLite) 모드에서는 아래 경로에 DB 파일이 자동 생성됩니다.

- **Windows**: `내 문서\ETA\Data\eta.db`
- **macOS**: `~/Documents/ETA/Data/eta.db`

### 2-3. 서버 DB 연결 설정 (MariaDB)

관리자가 `appsettings.json` 파일에 서버 정보를 기입해야 합니다.  
(일반 사용자는 건드릴 필요 없음)

## 3. 로그인 & DB 모드 전환

### 3-1. 로그인

1. 프로그램 실행 후 로그인 화면이 나타납니다.
2. **사번**과 **비밀번호**를 입력하고 **로그인** 버튼을 클릭합니다.
3. 비밀번호는 기본적으로 입력한 그대로 표시됩니다.

### 3-2. DB 모드 전환 (관리자 전용)

로그인 화면 왼쪽 상단에 보이지 않는 버튼이 있습니다.  
해당 영역을 클릭하면 **로컬 SQLite ↔ 서버 MariaDB** 모드를 전환할 수 있습니다.

| 모드 | 설명 |
|------|------|
| 로컬 SQLite | 인터넷 연결 없이 사용 가능, 개인 PC에 데이터 저장 |
| 서버 MariaDB | 여러 PC에서 동시 접속, 데이터 공유 가능 |

> 서버 DB에 연결이 안 될 경우 → 로컬 모드로 자동 폴백되지 않습니다.  
> 네트워크 상태 및 서버 주소를 확인하세요.

### 3-3. 비밀번호 변경

메인 화면 우측 상단 이름 영역을 클릭하면 **비밀번호 변경** 창이 열립니다.

## 4. 메인 화면 구성

```
┌──────────────────────────────────────────────────────────────────┐
│  [로고]  수질분석센터▾  비용부담금 관리▾  처리시설▾  관리자▾   [사용자명]  │
├──────────────────────────────────────────────────────────────────┤
│  [BT1]  [BT2]  [BT3]  [BT4]  [BT5]  [BT6]  [BT7]            │  ← 서브메뉴 버튼
├─────────────────────┬────────────────────────────────────────────┤
│                     │                  Show2                     │
│       Show1         ├────────────────────────────────────────────┤
│   (트리/메인 페이지)  │                  Show3                     │
│                     ├────────────────────────────────────────────┤
│                     │                  Show4                     │
└─────────────────────┴────────────────────────────────────────────┘
```

| 영역 | 역할 |
|------|------|
| Show1 | 트리 뷰 또는 메인 페이지 (좌측) |
| Show2 | 목록 / 신규 작성 폼 (우측 상단) |
| Show3 | 수정 폼 / 액션 패널 (우측 중간) |
| Show4 | 출력 보관함 / 분析항목 목록 (우측 하단) |

각 영역의 크기는 메뉴에 따라 자동으로 조절됩니다.

## 5. 메뉴별 기능 안내

### 5-1. 직원정보

**진입**: 상단 메뉴 → 수질분석센터 → 직원정보

| 기능 | 설명 |
|------|------|
| 직원 트리 | 좌측에 부서/직급별 직원 목록 표시 |
| 직원 추가 | BT3 → 신규 직원 등록 폼 표시 |
| 선택 삭제 | BT4 → 선택한 직원 삭제 (확인 필요) |
| 엑셀 내보내기 | BT5 → 전체 직원 목록을 xlsx로 저장 |
| 저장 | BT1 → 변경 내용 저장 |

### 5-2. 계약업체

**진입**: 상단 메뉴 → 수질분석센터 → 계약업체

| 기능 | 설명 |
|------|------|
| 계약 목록 | 좌측에 계약업체 트리 표시 |
| 업체 추가 | BT3 → 신규 계약업체 등록 |
| 계약 정보 | 계약기간, 금액, 담당자, 계약구분 등 관리 |
| 저장 | BT1 → 수정 내용 저장 |

### 5-3. 견적/의뢰서

**진입**: 상단 메뉴 → 수질분석센터 → 견적/의뢰서

4개 패널이 동시에 표시됩니다.

| 패널 | 내용 |
|------|------|
| Show1 (발행내역) | 과거 발행된 견적서 목록 (날짜 역순) |
| Show2 (세부내역/신규) | 선택한 견적 상세 또는 신규 작성 폼 |
| Show3 (확인) | 수량/금액 확인 패널 |
| Show4 (업체 목록) | 등록된 계약업체 목록 |

**주요 버튼**

| 버튼 | 동작 |
|------|------|
| BT1 저장 | 현재 폼 저장 |
| BT2 신규 작성 | 새 견적서 작성 폼으로 전환 |
| BT3 새로고침 | 발행내역 목록 갱신 |
| BT4 삭제 | 선택한 견적서 삭제 |

### 5-4. 시험성적서

**진입**: 상단 메뉴 → 수질분석센터 → 시험성적서

| 패널 | 내용 |
|------|------|
| Show1 | 업체/채취일 기준 트리 뷰 |
| Show2 | 선택한 시료의 분석결과 목록 |
| Show3 | 결과 수정 폼 또는 Excel 불러오기 패널 |
| Show4 | 출력 보관함 (저장된 엑셀/PDF 목록) |

**주요 버튼**

| 버튼 | 동작 |
|------|------|
| BT1 새로고침 | 트리 데이터 다시 불러오기 |
| BT2 CSV 저장 | 현재 표시 중인 결과를 CSV로 저장 |
| BT3 삭제 | 선택한 시료 삭제 |
| BT4 엑셀 출력 | 현재 시료 → 엑셀 성적서 출력 |
| BT5 PDF 출력 | 현재 시료 → PDF 성적서 출력 |
| BT6 일괄 엑셀 | 체크한 여러 시료 일괄 엑셀 출력 |
| BT7 자료TO측정인 | 분析 결과를 측정인 입력 창으로 전송 |

#### Excel로 분석 결과 불러오기

1. Show3 우측 상단 **Excel 불러오기** 토글을 활성화합니다.
2. Excel 파일을 드래그 앤 드랍하거나 파일 선택 버튼으로 불러옵니다.
3. 미리보기에서 충돌 항목을 확인 후 **저장** 버튼을 클릭합니다.

### 5-5. 보수요청

**진입**: 상단 메뉴 → 수질분석센터 → 보수요청

| 기능 | 설명 |

### 5-6. 물품구매

| 패널 | 내용 |

| 버튼 | 동작 |

### 5-7. 폐수배출업소

## 7. 자료TO측정인 기능

> **시험성적서 메뉴 → BT7 "자료TO측정인"** 버튼으로 접근

이 기능은 ETA에 입력된 분析 결과를 **측정인.kr 사이트**에 자동 입력하기 위한 보조 창입니다.

### 7-1. 창 구성

- **시료 목록**: 현재 선택된 업체의 분析 완료 시료 목록
- **입력 버튼**: 선택한 시료의 데이터를 측정인에 입력

### 7-2. 입력되는 항목

각 분석 항목별로 다음 값이 자동 입력됩니다.

| 항목 | 데이터 출처 |
|------|------------|
| 법적 기준 | 방류기준표 (DB) |
| 분석 결과 | 분析의뢰및결과 테이블 |
| 분析기기 | 드롭다운 클릭 선택 (측정인 사이트 기준) |
| 측정분析방법 | 분析정보 테이블 (Method 컬럼) |
| 분析장비 | 분析정보 테이블 (instrument 컬럼) |
| 분析자 | 로그인된 사용자 정보 |
| 분析일(시작일) | 채취일자 |
| 시작시간 | 채취시간 |
| 분析일(종료) | 분Analysis완료일자 |
| 종료시간 | (시료별 설정) |
| 시료용량 | (시료별 설정) |
| 비고 | (선택 입력) |

## 8. 데이터베이스 관리

### 8-1. 로컬 백업

- 로컬 모드: `내 문서\ETA\Data\eta.db` 파일을 복사하여 백업
- 정기적으로 백업 권장 (관리자 메뉴 → 백업하기)

### 8-2. 서버 DB (MariaDB)

- 서버: `1110s.synology.me:3306`
- DB명: `eta_db`
- 서버 DB는 Synology NAS에서 관리

### 8-3. DB 모드별 특성

| 항목 | 로컬 SQLite | 서버 MariaDB |
|------|------------|-------------|
| 속도 | 빠름 | 네트워크 속도에 따라 다름 |
| 동시 접속 | 불가 | 가능 |
| 백업 | 파일 복사 | NAS 자체 백업 |
| 오프라인 사용 | 가능 | 불가 |

## 9. 파일 출력 (엑셀 / PDF)

### 9-1. 시험성적서 출력

1. 시험성적서 메뉴 진입
2. 트리에서 출력할 시료 선택
3. **BT4 엑셀 출력** 또는 **BT5 PDF 출력** 클릭
4. 파일은 `내 문서\ETA\Data\Reports\` 폴더에 자동 저장
5. Show4 (출력 보관함)에서 저장된 파일 목록 확인 및 재열기 가능

### 9-2. 일괄 출력

1. 트리에서 각 시료 항목 왼쪽 체크박스 체크
2. **BT6 일괄 엑셀** 클릭 → 선택한 모든 시료 일괄 출력

### 9-3. 출력 보관함 (Show4)

- 저장된 Excel/PDF 파일 목록 자동 표시
- 파일명 클릭 → 해당 파일 열기
- 폴더 아이콘 → `Reports` 폴더 열기

## 10. 자주 묻는 질문 FAQ

**Q. 로그인이 안 됩니다.**  
A. 사번과 비밀번호를 확인하세요. 초기 비밀번호는 관리자에게 문의하세요.

**Q. 서버 DB 연결이 안 됩니다.**  
A. 사내 네트워크에 연결되어 있는지 확인하세요. VPN이 필요한 경우 VPN을 먼저 연결하세요.  
   그래도 안 되면 로그인 화면 좌상단을 클릭하여 로컬 모드로 전환 후 사용하세요.

**Q. 시험성적서 트리에 데이터가 안 보입니다.**  
A. BT1 **새로고침** 버튼을 클릭하거나, DB 모드를 확인하세요.

**Q. 엑셀 출력 파일이 어디 저장되나요?**  
A. `내 문서\ETA\Data\Reports\` 폴더에 저장됩니다. Show4 보관함에서도 확인 가능합니다.

**Q. 분析 결과를 수정했는데 저장이 안 됩니다.**  
A. 셀을 클릭한 후 값을 수정하고 **Enter** 키를 누르거나 다른 셀을 클릭하면 자동 저장됩니다.

**Q. PDF 출력 시 폰트가 깨집니다.**  
A. `내 문서\ETA\Assets\Fonts\` 폴더에 폰트 파일이 있는지 확인하세요.

**Q. 프로그램이 갑자기 종료됩니다.**  
A. 프로그램 폴더의 `crash.log` 파일 내용을 개발자에게 전달해 주세요.
